-   nextUI
-   canvas-confetti
-   axios
-   PokeAPI -> https://pokeapi.co

## Features

-   Pokemon List
-   Click Favorites by id.

## Installation

```sh
$ git clone https://github.com/dsagredo/project-pokemon-nextjs.git
$ cd project-pokemon-nextjs/
$ npm install
$ npm run dev
```
